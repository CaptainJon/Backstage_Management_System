# aether_gm_backstage

## 技术栈 Vue / Element-UI / Axios

### 项目依赖安装
```
npm install 
```

### 项目开发状态运行
```
npm run serve
```

### 项目打包
```
npm run build
```

### ESlint语法侦测
```
npm run lint
```
